* Primetrade

Primetrade is a full-stack task management application built using the MERN stack
It allows users to register,log in,and manage their personal tasks through a secure dashboard with full CRUD functionality.

* Features

* Authentication
- User registration and login
- JWT-based authentication
- Protected dashboard routes
- Secure logout flow

* Dashboard
- Fetches user profile from backend
- Create, read, update, and delete tasks
- Edit task title
- Mark tasks as Pending or Completed
- Search tasks by title
- Filter tasks by status
- Responsive UI with TailwindCSS

* Security
- Password hashing using bcrypt
- JWT authentication middleware
- Proper error handling
- CORS configured for production


* Tech Stack

* Frontend
- React.js (Vite)
- TailwindCSS
- Axios
- React Context API
- Deployed on Vercel

* Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Deployed on Render

* Database
- MongoDB Atlas
